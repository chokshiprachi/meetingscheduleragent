# MeetEase Pro AI

MeetEase Pro AI is a smart meeting scheduling web application built using HTML, CSS, and JavaScript.

It allows users to schedule meetings using natural language commands like:

Book tomorrow at 10am

The system detects the date and time, confirms the booking, simulates payment, and stores meetings in a dashboard.

------------------------------------------------------------

## Project Overview

MeetEase Pro AI is a frontend-only project that simulates AI behavior using JavaScript logic.

It demonstrates:
- Natural language date detection
- Automatic time selection
- Smart confirmation messages
- Payment simulation
- Reminder system
- Meeting dashboard

No backend or external APIs are used.

------------------------------------------------------------

## Features

- AI-based natural language scheduling
- Automatic detection of "tomorrow"
- Time extraction (10am, 2pm, etc.)
- Dynamic confirmation messages
- Payment simulation
- Reminder popup
- Dashboard with stored meetings
- LocalStorage data saving
- Clean and responsive UI

------------------------------------------------------------

## How AI Works

This project does not use real AI.

It uses:
- Keyword detection (example: "tomorrow")
- Regular expressions to extract time
- Conditional logic
- Automatic form filling

Example:

Input:
Book tomorrow at 10am

Process:
- Detects the word "tomorrow"
- Extracts time (10am)
- Sets date automatically
- Shows confirmation message
